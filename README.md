# AmbitionScraper - Career & Salary Data Analysis

## About the Project

This project is based on collecting and analyzing career and salary data from AmbitionBox. The aim is to understand salary trends and job patterns using real-world data.

---

## What I Did

* Collected data from AmbitionBox using web scraping
* Extracted details like job role, salary, company name, and experience
* Cleaned and organized the data using Pandas
* Performed analysis to understand salary distribution and trends
* Identified patterns based on roles and experience

---

## Tech Used

Python, Pandas, BeautifulSoup / Selenium

---

## Result

The project helped in understanding how salaries vary across different roles and experience levels. It also showed real-world job trends.

---

## Why this Project

This project helped me learn how to collect real data from websites and analyze it to get useful insights.

---

## How to Run

* Install required libraries
* Run the notebook
* Data will be processed and analyzed
